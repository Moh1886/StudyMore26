## 🚀 كيفية البدء

### الطريقة الأولى: الفتح البسيط
1. افتح ملف `auth.html` في متصفحك المفضل
2. أنشئ حساباً جديداً أو استخدم حساباً تجريبياً
3. ابدأ باستخدام جميع الميزات!

### الطريقة الثانية: خادم محلي (اختياري)
إذا كنت تريد تشغيل خادم محلي:
```bash
# إذا كان لديك Python
python -m http.server 8000

# أو إذا كان لديك Node.js
npx http-server
```

ثم افتح:
```
http://localhost:8000
```

## 🔒 الأمان والخصوصية

### ✅ معايير الأمان:
- جميع البيانات تُحفظ محلياً في جهاز المستخدم فقط
- لا تُرسل البيانات إلى أي خادم خارجي
- حماية من XSS والهجمات الشائعة
- التحقق من صحة جميع المدخلات

### ✅ الخصوصية:
- لا حاجة لإدخال بيانات حقيقية
- يمكن حذف كل البيانات بسهولة
- بيانات آمنة وسرية تماماً

---

## 💡 نصائح للاستخدام الفعّال

1. **ابدأ بالمواد الدراسية:** أولاً، أضف المواد التي تدرسها
2. **خطط جدولك:** حدد بدقة أوقات المذاكرة
3. **استخدم قائمة المهام:** اكتب كل المهام وحدد أولويتها
4. **استخدم مؤقت بوموداعو:** 25 دقيقة مذاكرة + 5 دقائق راحة
5. **راجع الإحصائيات:** تابع تقدمك باستمرار
