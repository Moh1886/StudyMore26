### 🎯 خطوات التشغيل

#### الطريقة الأولى: الفتح المباشر (الأسهل)

1. **افتح أحد الملفات التالية مباشرة في متصفحك:**
   - `welcome.html` ← 👈 اللينك الرئيسي
   - أو `auth.html` للدخول مباشرة
   - أو `index.html` إذا كنت ستسجل دخول

2. **يمكنك فتحها بـ:**
   - انقر بزر الماوس الأيمن على الملف واختر "Open with" ثم المتصفح
   - أو اسحب الملف وأفلته على نافذة المتصفح
   - أو انسخ المسار وأدخله في عنوان المتصفح

---

#### الطريقة الثانية: خادم محلي (اختياري)

إذا كنت ترغب بتشغيل خادم محلي:

**باستخدام Python (إذا كان مثبتاً):**
```bash
# اذهب إلى مجلد المشروع
cd "d:\بب\6g"

# شغل الخادم
python -m http.server 8000
```

ثم افتح: `http://localhost:8000/welcome.html`

**باستخدام Node.js (إذا كان مثبتاً):**
```bash
# تأكد من وجود http-server
npm install -g http-server

# شغل الخادم
http-server
```

#### الصفحات الأساسية:

1. **welcome.html** - صفحة الترحيب الرئيسية

   - الجدول الدراسي

4. **guide.html** - دليل البدء
   - شرح شامل للميزات
   - أسئلة وأجوبة شائعة

5. **about.html** - حول الموقع
   - معلومات عن المطور
   - تفاصيل الموقع والأمان
